# General Communication

Use this format for any internal message that doesn't fit a recurring status
report, a newsletter, or an FAQ answer.

## Sections
Produce exactly three sections, in this exact order, using these exact
heading labels (Heading 1 style):

1. Summary
2. Details
3. Next Steps

Each heading is followed by a short bulleted list of the relevant items.

## Closing line
Add one final paragraph after the Next Steps section containing exactly this
sentence, verbatim:

> Let us know if anything is unclear.
