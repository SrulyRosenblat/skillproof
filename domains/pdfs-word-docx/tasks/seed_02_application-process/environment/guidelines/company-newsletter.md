# Company Newsletter

Use this format for a company-wide announcement that celebrates recent wins,
spotlights a teammate, and previews upcoming social events.

## Sections
Produce exactly three sections, in this exact order, using these exact
heading labels (Heading 1 style):

1. Highlights
2. Team Spotlight
3. Upcoming Events

Each heading is followed by a short bulleted list of the relevant items.

## Closing line
Add one final paragraph after the Upcoming Events section containing exactly
this sentence, verbatim:

> Thanks for reading -- see you next month!
