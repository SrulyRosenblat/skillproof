# FAQ Answers

Use this format when someone asks a single direct question and wants a clear
answer, not a status update.

## Structure
- Restate the question as the heading (Heading 1 style).
- Give a direct one-paragraph answer below it.
- Optionally add a "Details" section with supporting bullets.

## Closing line
Add one final paragraph containing exactly this sentence, verbatim:

> Still have questions? Just ask.
