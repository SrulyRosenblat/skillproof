# General Internal Communication Guide

Use this format for internal communications that do not match a more specialized guide.

Required structure:

# <title>
Audience: <audience>
Date: <Month D, YYYY>

## Summary
Write 2 or 3 sentences.

## Key Changes
- Exactly 3 bullets.
- Each bullet must begin with a bold label like `- **Status:**`.

## Impact and Risks
- Exactly 2 bullets.
- One bullet must describe operational or customer impact.
- One bullet must describe risk, mitigation, or a decision that needs approval.

## Next Steps
1. Exactly 2 numbered items.

Style:

- Direct, concise, and appropriate for broad internal stakeholders or leadership.
- Surface the main change, business impact, and any request clearly.
- Do not use placeholder text.
