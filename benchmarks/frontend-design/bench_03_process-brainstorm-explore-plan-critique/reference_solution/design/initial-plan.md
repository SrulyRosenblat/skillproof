# Initial Design Plan — Aria

## Color
- Cream — #F4F1EA — page background
- Charcoal — #2B2A28 — body text
- Terracotta — #C1583C — accent, links, CTA button
- Warm White — #FBF9F4 — card surfaces

## Type
- Display: Playfair Display — hero headline and section titles
- Body: Source Sans Pro — paragraphs, labels, form copy

## Layout
A centered hero with headline and sub-line, followed by a three-column
feature grid, then a closing waitlist band.

```
+--------------------------------------+
|              header/nav              |
+--------------------------------------+
|            centered hero              |
|          headline + subline           |
+--------------------------------------+
|  feature  |  feature  |   feature    |
+--------------------------------------+
|          waitlist / signup            |
+--------------------------------------+
```

## Signature
Slug: pressure-dial
A circular pressure-gauge illustration sits beside the hero headline,
its needle resting in the 9 bar zone, as a decorative nod to the
machine's peak pressure.
